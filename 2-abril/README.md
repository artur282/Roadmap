# 🤖 Abril — IA/ML & GenAI

> _"La IA no reemplaza al ingeniero, lo potencia."_

## 🎯 Objetivo del mes

Demostrar capacidad para integrar inteligencia artificial en aplicaciones reales y productivas. Desde chatbots conversacionales hasta servidores MCP y pipelines RAG, cada proyecto expone IA como un servicio confiable y bien diseñado.

## 🧠 Habilidades que se desarrollan

- Integración de LLMs en aplicaciones backend
- Diseño de prompts y prompt engineering
- Construcción de servidores MCP (Model Context Protocol)
- Pipelines RAG (Retrieval Augmented Generation)
- Procesamiento de lenguaje natural (NLP)
- Embeddings, Vector DBs (Pinecone, Chroma, pgvector) y búsqueda semántica
- Gestión de contexto y memoria en chatbots

---

## 🔗 Cómo se conectan los proyectos

```
Semana 05: ConversAI
    │  Aprende: LangChain, prompts, memoria → base de IA
    ▼
Semana 06: MCPForge
    │  Aprende: MCP SDK, tool calling → IA como servicio
    ▼
Semana 07: DocuMind
    │  Aprende: RAG, embeddings, vector stores (Pinecone/pgvector) → Q&A sobre datos
    ▼
Semana 08: SentiScope
       Aprende: NLP, análisis de texto → IA práctica sin LLMs
```
